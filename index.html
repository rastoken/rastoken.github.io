How It Works

Technical architecture for sustainable token economy

Reward Engine

Off-chain system computes rewards based on test scores, votes, reputation, and complexity. Final token transfers sent via Solana blockchain.

Staking Model

Users send RasT to treasury wallet. Stakes tracked internally in SOS database. Returns or slashing applied based on system rules.

Voting System

Users sign votes with wallet signatures. Backend verifies signatures. Voting logic applied off-chain for scalability and reduced costs.

Security Sandbox

User programs run in isolated Docker containers or custom VMs. Never direct OS execution. Prevents exploits, malware, and unauthorized access.

Inflation Control

Fixed emission per epoch. No unlimited minting. Rewards distributed from pre-minted pool. Mint only for verified value.

On-Chain Verification

Solana blockchain confirms all final distributions. Transparent, immutable record. Trusted token transfers backed by Solana network security.
